# Portfolio Website - Project Brief

## Project Overview
Development of a professional portfolio website for a Construction Project Manager seeking new career opportunities. The site will serve as an online resume and showcase professional experience, skills, and projects.

## Background
**Client Profile:**
- Construction Project Manager with career experience in the field
- Basic HTML/CSS knowledge, limited JavaScript experience
- Learning to code but needs a professional online presence immediately
- Located in Omaha, NE

**Business Need:**
- Professional online presence for job applications
- Digital resume accessible to potential employers
- Showcase construction project portfolio
- Demonstrate technical aptitude and modern skills

## Project Goals
1. **Primary Goal:** Create a professional, modern portfolio website that effectively presents qualifications to potential employers
2. **Secondary Goal:** Build with maintainable technology that the client can update independently
3. **Tertiary Goal:** Deploy to GitHub Pages for free, reliable hosting with professional domain potential

## Success Criteria
- Clean, professional design that appeals to construction industry employers
- Mobile-responsive (works on all devices)
- Easy to maintain and update by client with basic HTML/CSS knowledge
- Successfully deployed to GitHub Pages
- Fast loading times (< 2 seconds)
- Includes all key professional information (experience, skills, projects, contact)

## Constraints
- **Technical Skills:** Client has basic HTML/CSS, limited JavaScript
- **Timeline:** Immediate need for job applications
- **Budget:** Free hosting solution required (GitHub Pages)
- **Maintenance:** Must be simple enough for client to maintain independently

## Technology Stack Decision
**Static HTML/CSS/JavaScript** chosen because:
- Matches client's existing skill level
- No build process or complex tooling
- Easy to understand and maintain
- Perfect for GitHub Pages deployment
- Fast, reliable, and professional appearance possible
- Can be enhanced later as client learns more

## Target Audience
- Hiring managers in construction industry
- Recruiters searching for Project Managers
- Construction companies seeking experienced PMs
- Professional network connections

## Key Deliverables
1. Fully functional portfolio website with sections:
   - Hero/Landing section with introduction
   - About Me section
   - Professional Experience timeline
   - Skills showcase
   - Projects portfolio
   - Contact information/form
2. Responsive design for all screen sizes
3. GitHub repository with clean, documented code
4. Deployment to GitHub Pages
5. Documentation for maintaining/updating content

## Timeline
- Planning & Design: Day 1
- Development: Day 1-2
- Content Integration: Day 2
- Testing & Refinement: Day 2
- Deployment: Day 2
- **Target Launch:** Within 2 days

## Risks & Mitigations
| Risk | Impact | Mitigation |
|------|--------|------------|
| Client unfamiliar with Git/GitHub | Medium | Provide clear documentation and simple workflow |
| Content not ready | High | Start with placeholder content, easy to update |
| Design not meeting expectations | Medium | Review design direction early, iterate quickly |
| Mobile responsiveness issues | Medium | Test on multiple devices before deployment |

## Out of Scope (Phase 1)
- Blog functionality
- Content Management System (CMS)
- Backend/database
- Advanced animations or complex JavaScript interactions
- Custom domain setup (can be added later)
- Contact form backend (will use mailto or external service)

## Future Enhancements (Phase 2+)
- Custom domain name
- Blog section for industry insights
- Interactive project gallery
- Testimonials section
- Analytics integration
- Contact form with backend

## Stakeholders
- **Primary:** Daniel Tso (Construction PM, site owner)
- **Secondary:** Potential employers and recruiters
