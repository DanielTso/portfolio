## How to Make Future Changes

### Update Your Work Experience

**File to Edit:** `index.html`

**Location:** Find the Experience section (around line 207)

**To Add New Job:**
1. Copy an existing job block (lines 241-269 for example)
2. Update the details:
   - Job title
   - Company name
   - Dates
   - Location
   - Responsibilities
   - Tags

**Example:**
```html
<div class="timeline-item">
  <div class="timeline-marker"></div>
  <div class="timeline-content">
    <div class="timeline-header">
      <h3 class="job-title">Your New Title</h3>
      <span class="job-badge current">Current</span> <!-- if current -->
    </div>
    <h4 class="company-name">Company Name</h4>
    <p class="job-meta">
      <i class="fas fa-calendar"></i> Start Date - End Date

    <ul class="job-responsibilities">
      <li>Responsibility 1</li>

      <span class="tag">Skill 1</span>

</div>
```

## Troubleshooting

### Common Issues

**Portfolio not updating after changes:**
- Wait 2-5 minutes for GitHub Pages to deploy
- Clear browser cache (Ctrl+Shift+Delete)
- Try incognito/private mode
- Check GitHub Actions: https://github.com/DanielTso/portfolio/actions

**Resume PDF not downloading:**
- Verify filename is exactly `Daniel_Tso_Resume.pdf`
- Check it's in `assets/documents/` folder
- Ensure file was committed and pushed to GitHub
- Wait a few minutes after pushing

**Images not showing:**
- Check file paths are correct (use relative paths)
- Verify images are in `assets/images/` folder
- Ensure images are committed to GitHub
- Check file extensions match code

**Can't push to GitHub:**
```bash
# Check if you're logged in
gh auth status

# Re-login if needed
gh auth login
```
