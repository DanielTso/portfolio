```bash

open index.html
```

No build process required - it's a static website!

### Deploy to GitHub Pages

1. Push to GitHub:
```bash
git add .
git commit -m "Initial portfolio commit"
git push origin main
```

2. Enable GitHub Pages:
   - Go to repository Settings
   - Navigate to "Pages" section
   - Select "main" branch as source
   - Click Save
   - Your site will be live at: `https://[username].github.io/[repo-name]`

## âœï¸ Updating Content

### Update Your Information

**Edit Work Experience:**
- Open `index.html`
- Find the `<section id="experience">` tag
- Update job titles, dates, companies, and descriptions

**Update Skills:**
- Locate `<section id="skills">` in `index.html`
- Add or remove skill items as needed

**Update Projects:**
- Find `<section id="projects">` in `index.html`
- Edit project cards with new information

### Update Resume PDF

1. Save your updated resume as PDF
2. Name it `Daniel_Tso_Resume.pdf`
3. Place in `assets/documents/` folder
4. Commit and push changes

### Update Photo

1. Add your professional photo to `assets/images/`
2. Recommended: Square format, at least 500x500px
3. Update the image path in `index.html` hero section
4. Commit and push changes

## ğŸ¨ Customizing Design

### Change Colors

Edit `css/variables.css` to change the color scheme:

```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --accent-color: #your-color;
}
```

### Change Fonts

1. Visit [Google Fonts](https://fonts.google.com)
2. Select your preferred fonts
3. Replace the Google Fonts link in `index.html`
4. Update font families in `css/variables.css`

## ğŸ› Troubleshooting

### Images Not Showing
- Check file paths are correct
- Ensure images are in `assets/images/` folder
- Verify image file extensions match code

### Styles Not Loading
- Check CSS file paths in `index.html`
- Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Verify CSS files exist in `css/` folder

### Resume Download Not Working
- Ensure PDF is in `assets/documents/` folder
- Check filename matches link in HTML
- Verify file uploaded to GitHub
